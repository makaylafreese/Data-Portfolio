# Restaurant Tipping Analysis

## Overview
Exploratory data analysis of restaurant tipping behavior using transaction-level data. This project examines how tip amount relates to bill size and dining conditions.

## Questions
- How strongly are tips associated with total bill size?
- Do tipping patterns differ by day of week or meal time?

## Tools Used
- Python
- pandas
- matplotlib
- seaborn

## Deliverables
- Jupyter notebook with EDA, visualizations, and key findings
